## Welcome to my new portfolio!

I wanted to redesign my old portfolio to make it more modern and dynamic, and I also wanted to have a blog-ish section
to share my projects, softwares and games. <br/>
I'm really happy with how it turned out, and I hope you like it too!

## Technologies used
I chosed to use [Astro](https://astro.build) because it's insanely fast and allow me to write my content in Markdown,
which makes it really easy to create and manage any posts.
I also used [Tailwind CSS](https://tailwindcss.com) for styling, which allowed me to create a beautiful and responsive design
without needing to write much custom CSS. <br/>
The usage of [GitHub Pages](https://pages.github.com/) for deployment made it easy to host my website for free and
keep it up-to-date with every push to the main branch.

## What I learned
Even though I don't have much experience with web development as I don't really like it, building this portfolio was a great learning experience. <br/>
I learned how to use Astro's component system to create reusable components to minimize code duplication,
and I was able to free myself from the constraints of the template I used in my old portfolio. <br/>
The ability to write content in Markdown and have it seamlessly integrated into my website was what made this project so rewarding.
I also enjoy working with GitHub Actions because I want to improve in CI/CD and automate my workflow as much as possible.

I'm really proud of this project and the flexibility it gives me to adapt and expand my website as I want,
and I look forward to sharing more content and projects in the future!
